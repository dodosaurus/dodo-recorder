## Getting Started

1. Fork the repository
2. Clone: `git clone https://github.com/yourusername/dodo-recorder.git`
3. Follow [Developer Setup](README.md#-developer-setup) instructions
4. Create a branch: `git checkout -b feature/your-feature-name`

## Development

Read [AGENTS.md](AGENTS.md) for coding standards and conventions.

Test locally:
```bash
npm run dev
```

## Submitting Changes

1. Push to your fork
2. Create a Pull Request
3. Describe your changes clearly

## Reporting Issues

Use GitHub Issues for bugs and feature requests. Search existing issues first.
